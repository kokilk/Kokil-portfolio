# AI Portfolio Website — Kokil TN

An interactive personal portfolio built from scratch using pure HTML, CSS, and JavaScript.  
Designed to showcase AI consulting work, projects, and real-world impact through a modern, dynamic interface.

---

## 🧠 What This Is

This is not a template-based portfolio.

It is a fully custom-built website that combines:
- AI consulting narrative
- interactive UI/UX
- real project experience
- dynamic animations and workflows

The goal was simple:  
**build a portfolio that feels like a product, not a page.**

---

## ⚙️ How It’s Built

This project is built using **only frontend technologies**:

- **HTML** → structure and content  
- **CSS** → styling, themes, animations, layout  
- **JavaScript** → interactivity, logic, and dynamic behavior  

No frameworks. No libraries for UI.

Everything from animations to navigation is handcrafted.

---

## ✨ Key Features

- Interactive landing section with animated canvas
- Smooth scrolling and section-based navigation
- Dark / light mode toggle
- Animated metrics and transitions
- Fully responsive design
- Structured sections for:
  - Experience
  - Projects
  - Skills
  - Education
  - Contact

---
